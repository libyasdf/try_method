候选样式表

浏览器支持候选样式表的话，会使用link元素title属性值生成候选样式表

一样的title值，属于同一组style link

如果不为样式表无标题，就是永久样式表，始终用于显示文档

style元素

应当始终设定type属性

style标签之间的部分为文档样式表或嵌入样式表

@import指令

与link的区别只在于：句法和「指令位置」

@import会使用每一个样式表,都会用于显示文档

媒体描述符
彩打
  • link标签里 media="print and (color)"
  • @import url() print and (color)
  • 彩色媒体都符合（color: 16）
  • (color) 这个媒体是彩色的吗？
  • link标签里 media="print and (color), screen and (color-depth: 8)"
  • @import url() print and (color), screen and (color-depth: 8)
